ETAPA 1: Valida a entrada do jogador

Para movimentar o jogador, utilizaremos as teclas W, A, S e D.

  • W: A tecla W será utilizada para mover para cima;
  • S: A tecla S será utilizada para mover para baixo;
  • A: A tecla A será utilizada para mover para a esquerda;
  • D: A tecla D será utilizada para mover para a direita;

Desta forma, faça um programa que peça para o usuário digitar qual direção ele deseja andar. Se o usuário digitar qualquer coisa diferente de w, a, s e d, o programa deve imprimir o texto abaixo e pedir novamente qual direção o jogador deseja andar:

Entrada do jogador inválida!
Você deve digitar a/s/d/w!

:::admonition{type=danger title="Importante"} As mensagens acima devem ser impressas utilizando o comando print :::

Note que, o usuário deve digitar as letras minúsculas, caso sejam digitados W, A, S e D em maiúscula, o programa também deve considerar como entrada inválida.

O programa deve finalizar somente quando o jogador inserir uma entrada válida. Ao final o programa deve imprimir: Entrada válida

Exemplo 1: 

> a
Entrada válida

Exemplo 2: 

> A
Entrada do jogador inválida!
Você deve digitar a/s/d/w!
> X
Entrada do jogador inválida!
Você deve digitar a/s/d/w!
> ?
Entrada do jogador inválida!
Você deve digitar a/s/d/w!
> w
Entrada válida

:::
